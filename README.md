# 🍕 Sistema de Pedidos para Pizzaria

## 📖 Sobre o Projeto

Este projeto é um sistema de pedidos para pizzaria desenvolvido utilizando HTML, CSS e JavaScript puro.

O objetivo foi criar uma experiência simples e intuitiva para que os clientes possam visualizar o cardápio, adicionar produtos à sacola e finalizar seus pedidos diretamente pelo WhatsApp.

## ✨ Funcionalidades

* Visualização do cardápio de pizzas
* Filtro por categorias
* Sacola de pedidos dinâmica
* Cálculo automático do valor total
* Integração com WhatsApp
* Cadastro de:

  * Nome
  * Endereço
  * Telefone
  * Observações
* Layout responsivo para desktop e mobile

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Lucide Icons

## 🚀 Funcionalidades Implementadas

* Catálogo de pizzas
* Sistema de sacola
* Cálculo automático do total
* Integração com WhatsApp
* Limpeza automática dos campos após o envio
* Design responsivo

## 📱 Demonstração

Após selecionar os produtos desejados, o usuário pode finalizar o pedido e enviar automaticamente as informações para o WhatsApp da pizzaria.
